安装与配置 Tiny Core Linux

Apr 09, 2023
阅读时长: 1 分钟

安装

配置DNS服务器

1

sudo printf "nameserver <DNS服务器IP>" > /etc/resolv.conf

安装安装程序

1

tce-load -wi tc-install

安装

1

sudo tc-install.sh
